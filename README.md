# sistema-chat
Este sistema de chat esta siendo desarrollado para aprender a implementar autenticación y autorización en aplicaciones web, utlizando JWT para el manejo de sesiones. Actualmente el backend esta siendo desarrollado en FastAPI y el frontend en React.

## Tecnologías utilizadas

- Python 3.12
- FastAPI
- SQLAlchemy
- JWT
- React 19
- Tailwind CSS
